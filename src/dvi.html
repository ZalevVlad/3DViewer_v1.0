Внешний вид приложения

Описание рабочей области

  • В поле Object отображается количество вершин (Vertexes) и количество рисуемых граней (Edges) объекта.
  • Поле Move предназначено для перемещения объекта.
  • Поле Rotate предназначено для поворота объекта вокруг его осей.

  • Поле Scale предназначено для изменения размера объекта.

  • Для переключения отображения координатных осей используется переключатель (Show axes).

  • Кнопка (Save IMAGE) сохраняет рендер изображения объекта.
  • Кнопка (Record GIF) стартует 5-секундную запись GIF.
  • Клик по цветовой палитре позволяет выбрать соответствующий цвет.
  • В поле Projection происходит выбор проекции изображения
  • В поле Edge Type переключается вид линий объекта с целого на штрихованный.
  • Используйте параметр Size для изменения толщины линий или размера рисуемых вершин.
  • В поле Vertex Type переключается вид рисуемых вершин (не рисовать/ круг/ квадрат).
